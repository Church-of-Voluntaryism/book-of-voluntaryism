# The Book of Voluntaryism

The foundational scripture of the Church of Voluntaryism, presented in a traditional, Bible-like format.

## About

This book contains the sacred texts and philosophical foundations of Voluntaryism:

- **Book I: Genesis** - The creation story and foundation of voluntary principles
- **Book II: Principles** - The moral foundations and guiding principles  
- **Book III: Parables** - Stories of wisdom and guidance
- **Book IV: Proverbs** - Teachings for righteous living

## Design

The book is styled with a conservative, Bible-like aesthetic featuring:
- Classic serif typography (Crimson Text, Georgia, Garamond)
- Numbered verses in biblical format
- Earth-tone color scheme with gold accents
- Traditional chapter and verse structure
- Print-optimized layout

## Publishing

This repository automatically publishes to GitHub Pages using Jekyll. Any changes pushed to the `main` branch will trigger a rebuild and deployment.

### Local Development

To build and preview the book locally:

```bash
# Install dependencies
bundle install

# Serve locally
bundle exec jekyll serve

# View at http://localhost:4000
```
